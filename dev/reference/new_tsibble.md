# Create a subclass of a tsibble

Create a subclass of a tsibble

## Usage

``` r
new_tsibble(x, ..., class = NULL)
```

## Arguments

- x:

  A `tbl_ts`, required.

- ...:

  Name-value pairs defining new attributes other than a tsibble.

- class:

  Subclasses to assign to the new object, default: none.
